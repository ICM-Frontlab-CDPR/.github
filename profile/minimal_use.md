# Minimal Git Usage

This document presents the essential Git commands to:
- version code locally
- synchronize code with a remote repository

## 1. Local version control

The three main commands to track and save code changes:

```bash
git status
git add .
git commit -m "Commit message"
```

## 2. Synchronized code sharing (remote)

The two main commands to synchronize with a remote repository:

```
git pull
git push
```

git pull : fetch and merge remote changes

git push : send local commits to the remote repository

git fetch : fetch remote changes without applying them
